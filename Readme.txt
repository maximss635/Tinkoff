Запуск: python main.py

Ввод размера поля и количесвта бомб из стандартного потока вывода
Далее общение через псвевдо-командную строку
help - посмотреть список команд
Возможно сохранение игры (команды save/load)
При сохранении вся информация об игре записывается в файл field.backup, шифруясь на rsa
Так как использовать сторонние библиотеки нельзя, алгоритм rsa реализован вручную (my_rsa.py)
Ключи для RSA заданы статически в коде - открытый (1594536295583, 2321726850343), закрытый - (34862323487, 2321726850343)
Благодаря шифрованию обычный пользователь не может прочитать field.backup
